# Super-Bubble-Builder
<p>Releases page: https://github.com/BlackSheepBoy69/Super-Bubble-Builder/releases/</p>
<p>Create stylish individual bubbles for your retro games on PlayStation Vita using a PC with Windows XP or higher.</p>
<p><img src="/Media/screen_01.jpg" width="900" title="screen-09"></p>
<h2>Instructions</h2>
<p>Super Bubble Builder comes in a .zip file. Inside is Super Bubble Builder's folder. The 'assets' subfolder has some sample images, as well as a png palette reducer. To make a bubble with this app, you can either use all 3 images, or you can just use icon0.png, but they must be .png and they must be the right size. [DRok17's bubble builder](https://github.com/DRok17/Multi-Builder) is an alternative that can use jpg and can resize the image for you, but it has some restrictions, and uses the terminal as an interface, and doesn't support nintendo DS yet, as of 6/9/2026.</p>
<p><img src="/Media/screen_02.jpg" width="900" title="screen-09"></p>
<p>The image sizes you'll want are:</p>
<p>--------------------------------------------------</p>
<p>128x128 png with 256 palette named icon0.png</p>
<p>--------------------------------------------------</p>
<p>380x158 png with 256 palette named startup.png (optional)</p>
<p>840x500 png with 256 palette named bg.png (optional)</p>
<p>--------------------------------------------------</p>
<p>The images must be placed in the assets folder before you run Super Bubble Builder. If startup.png or bg.png are missing, it will proceed with just icon0, and Vita will generate the rainbow background. You can make a png's palette 256 by dragging them to a batch script in the assets folder called 'Drag PNG here to reduce palette to 256.bat'</p>

<p>Releases page: https://github.com/BlackSheepBoy69/Super-Bubble-Builder/releases</p>

<p>User 'After-Pop-2000' has reported that "If the images in the GUI are screwed up, right click the program and choose: Properties > Compatibility > Change high DPI settings, enable Override high DPI scaling behavior performed by: and set it to <b>System</b>"</p>

<p>If you are experiencing this issue, please post about it in the issues section, it would be good to know more about it.</p>
